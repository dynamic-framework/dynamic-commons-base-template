# Dynamic Commons Base Template

Este archivo reemplaza la documentación estática del proyecto. Puedes agregar contenido directamente aquí sin depender de @storybook/blocks.

---

## Propósito

- Proveer una base robusta para bibliotecas de componentes escalables y mantenibles.
- Fomentar la consistencia y las mejores prácticas entre equipos.
- Habilitar la creación rápida de prototipos y documentación con Storybook.

---

## Cómo usar

### 1. Clona este repositorio e instala las dependencias

```sh
git clone <repo-url>
cd dynamic-commons-base-template
npm install
```

### 2. Inicia Storybook para explorar y desarrollar componentes

```sh
npm run storybook
```
