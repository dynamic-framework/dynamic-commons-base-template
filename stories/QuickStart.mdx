## Purpose

- Provide a robust foundation for scalable and maintainable component libraries.
- Promote consistency and best practices across teams.
- Enable rapid prototyping and documentation with Storybook.

---

## How to use

### 1. Clone this repository and install dependencies

```sh
git clone <repo-url>
cd dynamic-commons-base-template
npm install
```

### 2. Start Storybook to explore and develop components

```sh
npm run storybook
```
